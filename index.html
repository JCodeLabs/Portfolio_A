Acerca de mí.

Tras tres años trabajando en calidad en sectores como el Agroindustrial y el Retail, tomé la decisión de cambiar radicalmente mi carrera para seguir mi pasión por la tecnología y la programación. Me sumergí en el aprendizaje autodidacta de las principales tecnologías y lenguajes del desarrollo web. Desde entonces, he dedicado innumerables horas a perfeccionar mis habilidades en estas áreas, demostrando mi progreso a través de una serie de proyectos personales.

Me apasiona dar vida a diseños o crear interfaces de usuario desde cero, siempre con el objetivo de que sean visualmente atractivas, intuitivas y accesibles para todos. Mi constante búsqueda es elevar el estándar de diseño y calidad en cada proyecto que emprendo.

  • Mis habilidades son: HTML5, CSS3, JavaScript, React, Git & GitHub, Bootstrap, Tailwind & Figma.

Mis servicios.

Desarrollo de interfaces de usuario

Creación de interfaces de usuario visualmente atractivas y responsivas para garantizar una experiencia óptima en diferentes dispositivos y pantallas.

Implementación de diseños

Transformación de diseños proporcionados por diseñadores en interfaces web funcionales utilizando HTML, CSS y JavaScript.

Optimización del rendimiento

Garantizar que las páginas web se carguen rápidamente y funcionen eficientemente mediante la optimización del rendimiento y la experiencia del usuario.

Mantenimiento y actualización de sitios web

Realización de mantenimiento regular y actualizaciones en sitios web existentes para garantizar su funcionalidad y seguridad continua.

Colaboración con equipos multidisciplinarios

Colaboro estrechamente con diseñadores, desarrolladores backend y otros profesionales para ejecutar proyectos web de forma efectiva y satisfacer las necesidades del cliente.

Solución de problemas y depuración

Identificación y resolución de problemas técnicos y errores básicos en el código frontend para garantizar un funcionamiento óptimo de los sitios web.
